## Classification

**`webview-runtime-failure`** — GTK cannot initialize without a display server.

This is an environmental limitation, not an application bug. Tauri/tao requires a functioning GTK display server (X11 or Wayland) to create its event loop and window. WSL2 without a display server or working Xvfb cannot provide this.
