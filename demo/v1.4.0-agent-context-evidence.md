# v1.4.0 Agent Context Evidence

## Summary

v1.4.0 introduces Agent Context v2: a schema-versioned, content-policy-aware DTO that provides structured repository metadata to docs-agent and bugfix-agent.

## Agent Context v2 Schema

| Field | Type | Content |
|-------|------|---------|
| schema_version | string | "agent-context-v2" |
| branch | string | Current branch |
| head_short_sha | string | HEAD SHA (short) |
| dirty | bool | Working tree dirty |
| changed_files | array | Path, status, file_kind, risk (no contents) |
| risk_summary | object | Counts by risk category |
| diff_stat | object | files_changed, insertions, deletions |
| commit_groups | array | Scope, type, risk, suggested title |
| semantic_proposal | object | Title, scope, type, risk, confidence (no body) |
| recent_commit_subjects | array | Last 5 commit messages |
| provider | string | "deterministic-heuristic" |
| content_policy | object | All exclusion flags |
