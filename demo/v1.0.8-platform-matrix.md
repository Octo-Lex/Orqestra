## Key Insight

**Compile success is not platform support.** All three platforms compile successfully in CI, but only Windows has:
1. A standard installer artifact
2. SHA256 checksum verification
3. Local smoke test evidence
4. Manifest and documentation that agree

Linux and macOS have proven compilation but lack bundles, checksums, smoke evidence, and signing. They are not promoted.

## Guiding Rule

> No platform may be promoted unless it has an artifact, checksum, smoke evidence, manifest entry, and release documentation that all agree.
