## Agent Context v2 Summary

| What Changed | Before (v1.3.1) | After (v1.4.0) |
|-------------|-----------------|----------------|
| Context struct | `AgentGitContext` (flat lists) | `AgentContextV2` (structured, schema-versioned) |
| Schema version | None | `agent-context-v2` |
| Commit groups | Not included | Included with scope, type, risk, title |
| Proposal summary | Not included | Summary-only (no body field) |
| Content policy | Implicit | Explicit `ContentPolicy` struct |
| Context status | Silent `.ok().unwrap_or({})` | Explicit `available`/`unavailable` with error code |
| Agent constraints | `auto_commit: false` only | `review_only`, `auto_commit`, `auto_apply` all explicit |
| Diagnostics UI | None | `AgentContextPanel` + `AgentDiagnosticsPanel` |
