# Orqestra

Orqestra is a local-first project management tool with AI-native workflows.

It combines a Markdown-based task system, semantic commit analysis, CRDT-powered sync,
and multi-agent AI assistance into a single desktop application built with Tauri and React.

## Features

- **Markdown Roadmap** — Tasks as `.md` files with YAML frontmatter
- **AI Agents** — Docs and bugfix agents propose edits for human review
- **Semantic Commits** — AI-powered commit intent extraction
- **CRDT Sync** — Loro-based real-time collaboration
- **Graph Store** — Vector search across project knowledge
