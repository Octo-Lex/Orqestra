## Protocol v1

All messages: `{ protocol_version: 1, message_id: uuid, ... }`

Client → Server: join, delta, snapshot, leave
Server → Client: welcome, delta, peer_join, peer_leave, ack, error

Idempotency: message_id tracked per workspace (LRU 10,000). Duplicates → ack without reapplying.
