## Key Design Decisions

1. **Provider enum, not strings** — `GitProvider` is the sole source of provider labels
2. **Read-only diagnostics only** — `build_provider_report()` never mutates the repository
3. **Mutating ops registered, not executed** — staging/commit/push/pull/merge show `executed_in_diagnostics: false`
4. **Response wrappers for empty results** — `RecentCommitsResult` and `DiffStatResult` carry provider even on empty data
5. **Commit creation = gix-hybrid** — tree-from-index uses `git write-tree` CLI, so it's hybrid not fully native

## No-Mutation Guarantee

Test `git_status_unchanged_after_diagnostics` verifies:
- dirty flag unchanged
- staged_count unchanged
- unstaged_count unchanged
- untracked_count unchanged
